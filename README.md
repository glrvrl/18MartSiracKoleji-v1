## 📖 Hakkında

**Özel On Sekiz Mart Siraç İlkokulu**, 2007 yılında **Çanakkale/Kepez** ilçesinde kurulmuş, manevi değerlerle modern eğitim anlayışını harmanlayan özel bir ilkokuldur. Okulumuz, okul öncesi ve ilkokul kademelerinde eğitim vermektedir.

Bu proje, okulumuzun **resmi web sitesinin** kaynak kodlarını içermektedir. Site; öğrenci ön kayıt, personel başvuru, iletişim ve tanıtım amaçlı olarak geliştirilmiştir.

> 🔗 **Canlı Site:** [https://18martsirackoleji.k12.tr](https://18martsirackoleji.k12.tr)

## 📧 E-posta Entegrasyonu

Sitedeki 3 form, **PHPMailer** kütüphanesi ve **Gmail SMTP** kullanılarak e-posta gönderir:

| Form | PHP Dosyası | Açıklama |
|------|-------------|----------|
| Öğrenci Ön Kayıt | `onkayit_submit.php` | Öğrenci ve veli bilgileri |
| Personel Başvuru | `personalRandevu_submit.php` | İş başvurusu + CV/fotoğraf eki |
| İletişim | `iletisim_submit.php` | Ziyaretçi mesajları |

### ⚙️ Kurulum

PHP dosyalarındaki SMTP ayarlarını kendi bilgilerinizle güncelleyin:

```php
$aliciEmail     = 'sizin-email@gmail.com';
$gmailKullanici = 'sizin-email@gmail.com';
$gmailSifre     = 'APP_PASSWORD';  // Gmail App Password
```

> **Gmail App Password** oluşturmak için:
> 1. [Google Hesap Güvenliği](https://myaccount.google.com/security) sayfasına gidin
> 2. 2 Adımlı Doğrulamayı açın
> 3. **Uygulama Şifreleri** bölümünden yeni bir şifre oluşturun

## 🚀 Yerel Geliştirme

Projeyi yerel ortamda çalıştırmak için:

```bash
# 1. Repoyu klonlayın
git clone https://github.com/AHMETfarah22/SiracKolejiIlkokulu.git

# 2. Proje klasörüne gidin
cd SiracKolejiIlkokulu/home

# 3. PHP sunucusunu başlatın
php -S localhost:8080

# 4. Tarayıcıda açın
# http://localhost:8080
```

> ⚠️ PHP formlarının çalışması için sayfaları `http://localhost:8080` üzerinden açmanız gerekir. Dosyaları doğrudan tarayıcıda açmak (`file://`) PHP işlemlerini çalıştırmaz.
